# AI Doctor 赛道市场与监管调研报告

> 范围：中国 "AI 临床决策辅助 / AI 医疗" 赛道商业化可行性
> 视角：一家轻资产、无自有临床团队、依赖外部合作、核心资产为"多 Agent 共识引擎"的早期产研型公司
> 日期：2026-06

---

## 0. 一页结论（TL;DR）

1. **大盘性感，细分降温**：AI 医疗整体高速增长（2023 年约 107 亿元 → 2028 预测 976 亿元），但临床决策辅助（CDSS）细分 2024 年增速骤降到 **5.4%**，行业已从"技术叙事"进入"价值兑现"期——**纯卖模型/技术撑不住**。
2. **监管是诊断线的死结，但不是整个方向的死结**：给诊断结论 / 读影像 → 必走 NMPA **三类证**（截至 2024.6 全国仅 92 款影像类获批，原 100+ 玩家约 90% 被三类证时代淘汰）；不给诊断结论的院内效率/咨询类 → 可不走器械审批。**门槛卡在"诊断"两个字上，不在"AI 医疗"四个字上。**
3. **to B2B2C（医院再向患者收费）基本走不通**：73.9% 三甲已部署 AI 影像，但**仅 4.4% 向患者收费**；医保明确 AI 不单独向患者收费。
4. **C 端健康入口已被巨头焊死**：蚂蚁阿福（月活 1500 万、30 万医生、5000 家医院、医+药+险闭环）占据"健康咨询"心智，且变现靠流量+药险生态，不靠 AI 本身——**正面打 C 端 = 送死**。
5. **全行业长期亏损**：推想、数坤、讯飞医疗、鹰瞳普遍持续亏损、应收账款高企、回款慢，头部到 2025 年才"接近"盈亏平衡（注：此为预测，未见已实现确认）。
6. **对本公司的现实位置**：避开"C 端咨询红海"和"辅助诊断监管死路"，落在**"复杂决策 / 第二意见 / 院内效率"这一相对空白、且巨头标准化打法不灵的中间层**，并让差异化锚定在"多 Agent 共识 = 可见的多专家会诊"。

## 1. 市场规模与增速

| 指标 | 数据 | 来源 |
|---|---|---|
| AI 医疗整体规模 | 2019 年 27 亿 → 2023 年 107 亿元；2028 预测 976 亿元（占 AI 行业 15.4%） | 新浪财经 / 惠每 |
| CDSS 细分增速 | 2024 年同比 **仅增 5.4%**，明显放缓 | 同上 |
| 行业阶段判断 | 国金证券：AI 医疗已进入"商业化加速期……纯粹的技术赋能故事已难以维系企业长期发展" | 国金证券研报 |
| CDSS 三大成熟应用 | 辅助诊断、治疗方案规划/评估、医嘱输入/电子处方 | 行业研究 |

**含义**：大盘仍在涨，但钱开始向"能把技术与具体临床场景深度融合、且能量化产品价值"的企业集中。对无临床场景闭环的轻资产团队，单纯卖模型/技术难以立足。**叙事必须从"我们技术强"转向"我们解决了谁的什么、省了多少钱/担了多少责"。**

## 2. 监管准入门槛（核心壁垒）

### 2.1 三类器械审批 = 诊断线的死墙
- AI 辅助诊断软件（SaMD）多走 NMPA **三类器械**审批。
- 截至 2024.6，全国仅 **92 款** AI 影像诊断软件获三类证；头部企业各持 10+ 证。
- 原 100 余家竞争者中约 **90% 被三类证时代淘汰**。
- 申报需综合论证：自研软件研究、网络安全研究、AI 算法研究（含**数据质控、算法性能验证**）、可用性工程研究等多域重资料。

> **对本公司**：无临床团队 + 因果验证弱 = 难以独立完成"算法性能验证"与临床评价。三类证路径对轻资产团队几乎不可承受。

### 2.2 数据合规
- 涉及个人信息保护法、医疗数据出境限制、医院 HIS/PACS 数据接入权限。
- 私有化部署 + 数据不出院是医院侧的硬性前提，进一步抬高交付成本。

### 2.3 一个被证伪、不能当依据的传言 ⚠️
- 网传"2024.11 医保局将 AI 辅助诊断纳入可收费/放射类扩展项"——本次调研经多轮对抗验证**被否决（证据不足）**。
- 实际可确证的政策立场是：**AI 不单独向患者收费**。这反而构成 to B2B2C 收费路径的障碍。**对外汇报请勿引用该传言。**

### 3.3 集中度
- 中国医疗 AI 市场**高度分散**：2022 年讯飞医疗以 8.7% 收入份额第一，**CR5 仅 21.6%**。
- 无赢家通吃 → 细分场景仍存切入缝隙，但份额易被侵蚀。

## 4. to B 卖给医院的真实痛点

- **本质是 2B 生意，最大买单方是医院**，但付费意愿弱（73.9% 三甲部署 AI 影像 / 仅 4.4% 向患者收费）。
- **双重关卡**：医院采购流程（科室→信息科→院办→招标） + 物价/医保定价。
- **AI 还会打破医院现有利益格局**，作为新事物推进阻力大。
- **回款慢**：头部应收账款高企、大量逾期，对现金流是持续压力。
- **门槛效应**：早期公司无装机案例、无三甲背书，进不了招标门槛——这是头部"营收高但持续亏损"的根因之一。

## 5. 多 Agent 架构在临床决策的价值与风险

> 本节为分析判断，调研未产出可直接验证的第三方证据，标注为开放问题。

**潜在价值**：临床复杂决策（疑难病、MDT 多学科会诊、复杂用药）本质就是"多专家辩论求方案"，与多 Agent 共识机制**天然同构**；其差异化不在"给答案"，而在"呈现分歧、理由、不确定性"。

**风险**：
1. **相关性错误（correlated errors）**：若 Agent 共享同一底座模型，可能"一起自信地错"。需异构底座 + 独立知识库/Skill 缓解（本公司已部分实现，建议用"错误重合度"指标实测坐实）。
2. **假性收敛 / sycophancy**：收敛若靠"说服力"而非"正确性"，在医疗里致命，必须有外部 grounding（指南/文献/外部专家）兜底。
3. **责任与可解释性**：一旦输出被解读为诊断结论，仍可能触发器械监管。**产品形态须严守"呈现推理/选项、不给诊断结论"。**

## 6. 对本公司的现实切入点

| 切入点 | 是否需三类证 | 是否需临床团队 | 付费逻辑 | 匹配度 |
|---|:--:|:--:|---|:--:|
| **复杂/疑难"可见的多专家第二意见"** | 否（呈现推理，不给诊断） | 弱（靠外部合作专家+知识库） | 高价值低频，差异化清晰 | ★★★★★ |
| 病历质控 / DRG-DIP 编码 | 否（信息化软件） | 弱 | 医院省钱/避罚，经济动机强 | ★★★★☆（但偏效率软件，非 AI 原生） |
| 随访 / 患者教育 | 否 | 中 | 科室运营预算 | ★★★☆ |
| 用药审查 | ⚠️ 可能触二类 | 中 | 药学部，但有合规边界 | ★★★ |
| 辅助诊断（影像/结论） | **三类，必死** | 强 | — | ✗ |
| C 端健康咨询 | 否 | — | 被阿福生态焊死 | ✗ |

**推荐**：以**"复杂问题的多 Agent 共识第二意见"**为切入点——它把公司唯一真护城河（多 Agent 共识）放在聚光灯下，绕开监管（不给诊断结论），避开阿福（不打 C 端流量战），且不依赖自有临床团队。详见配套《Demo 设计文档》。

## 7. 待核实的开放问题（建议法务/注册专项）

1. 哪些院内/咨询场景可被认定为非器械或仅二类，从而绕开三类审批？需 NMPA 分类界定文件 + 已落地非器械 AI 案例佐证。
2. 无自有临床团队时，外部医院合作能否替代自建临床团队完成注册所需的回顾性/前瞻性数据与临床评价？
3. 在"AI 不单独向患者收费"前提下，目标场景真实付费方（科室预算 vs 信息化采购 vs 设备打包）与可量化 ROI、销售周期？
4. 多 Agent 架构在临床决策中的责任归属与监管边界，需专项法务调研。
